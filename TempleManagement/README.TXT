=======================================
Quick Startup Guide for Temple App v1.0
=======================================

Double click on "Temple App v1.0" to start. A database folder will be generated.

If you want to restart the database from beginning, delete this templeDB folder. It will be regenerated fresh and blank the next time you open the program. It will still have:
	1. one admin account 
	2. one dummy member account (which can be deleted)


================
Features in v1.0
================
Following modules fully work:
1. Members
2. Pai Wei
3. Users
4. Logs

Following modules does not work fully:
1. Donation
2. Volunteers
3. Accommodation
